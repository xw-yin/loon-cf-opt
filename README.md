# ⚡️ Loon Cloudflare 优选 IP 自动注入插件

这是一个专为 iOS 网络代理工具 **Loon** 设计的高级插件系统。它能够自动、定时维护 Cloudflare 的最快“优选 IP”缓存，并在你更新节点订阅时，将这些高性能的优选 IP 动态、轮询地注入到你的 VLESS / Trojan 代理节点中，实现代理连接速度起飞！

---

## 🌟 核心特性

- 🕒 **定时自动抓取**：内置的 Cron 定时任务每 12 小时自动运行一次，从互联网最新且经过测速检验的优选 IP 源拉取 IP，存储到本地缓存中。
- 🔄 **智能轮询注入**：在你手动或自动更新订阅时，重写脚本会自动拦截响应，并用最新缓存的优选 IP 轮询替换各个节点的网络连接地址，防止单 IP 过载。
- 🛡️ **证书与凭证安全**：整个流程在本地（iOS 设备）运行，不会向任何第三方泄露你的真实 UUID 或订阅密钥。
- 🚀 **性能免配置**：全自动打通网络握手，省去本地电脑测速并手动填写的繁琐步骤。

---

## 📂 文件目录说明

- **`cf_cron.js`**：定时任务脚本，负责获取最优质的 CF 优选 IP 并写入 Loon 持久化数据库。
- **`cf_sub.js`**：订阅重写脚本，解析 VLESS/Trojan 订阅内容，动态轮询替换节点 IP，并重新 Base64 编码输出。
- **`cf_opt.plugin`**：Loon 插件配置文件，包含所有的路由拦截定义与定时器属性。

## 🛠 安装与使用方法

### 第一步：开启 MitM 并信任证书
由于很多订阅链接和优选 IP API 是 HTTPS 链接，Loon 需要通过解密流量才能正常拦截：
1. 打开 Loon 客户端，进入 **“配置” -> “证书管理”**。
2. 生成 CA 证书，并根据 Loon 提示前往 iOS **系统设置 -> 通用 -> 关于本机 -> 证书信任设置** 中，对 Loon 证书开启**始终信任**。

### 第二步：安装插件
1. 打开 Loon，进入 **“配置” -> “插件”**，点击右上角 `+`，选择 **“从链接安装”**。
2. 填入你 GitHub 仓库上生成的 `.plugin` 文件的 `Raw` 链接：
   ```url
   https://raw.githubusercontent.com/xw-yin/loon-cf-opt/main/cf_opt.plugin
   ```
3. 点击保存并启用插件。

### 第三步：修改订阅匹配规则 (关键)
为了让 Loon 正确拦截你的订阅链接，请编辑刚刚导入的插件：
1. 将 `cf_opt.plugin` 中 `http-response` 对应的正则匹配表达式：
   ```ini
   http-response ^https?:\/\/your-sub-link\.com\/sub ...
   ```
   修改为你**自己真实的节点订阅链接域名与路径**。
2. 将 `[MITM]` 块下的 `hostname` 修改为你**真实的订阅解析域名**。

### 第四步：手动初始化并开始使用
1. 在 Loon 的 **“配置” -> “脚本”** 中找到 **“CF优选IP定时抓取”** 脚本，点击右侧的 **“手动执行”**。
2. 如果看到系统弹窗通知 `IP 缓存更新成功`，证明优选 IP 已成功拉取并缓存至本地！
3. 回到 Loon 节点列表，手动点击更新你的订阅。更新成功后查看节点详情，你会发现其连接地址已自动变成了最新最快的优选 IP。
